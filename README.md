Kaggle 및 Dacon의 완료된 대회 및 진행 중인 대회에 참가하여 기초적인 Python 및 데이터 분석 실력을 쌓았음

1. [ml](/ml) : 머신러닝 분류/회귀 문제와 관련된 대회

3. [dl_text](/dl_text) : RNN 등의 기초 텍스트 분류 모델부터 BERT까지의 사전 훈련된 모델을 사용하여 텍스트 분류 대회 참가
